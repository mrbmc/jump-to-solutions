# Jump to Solutions

A satirical UX problem solver inspired by the legendary printer scene from *Office Space*, but with a modern tech twist.

## What Is This?

Ever had a UX problem and immediately thought, "We should add an AI agent!" or "Dark mode will fix this!" or "Let's just A/B test it into oblivion"?

**Jump to Solutions** is a hilariously absurd game that smashes through the most generic, nonsensical tech solutions to your problems. Enter your UX challenge, click the grid, and watch it spin through a curated collection of buzzword-laden fixes before landing on the *obvious* solution.

## How to Play

1. **Click "START"** to begin
2. **Describe your UX problem** in the textarea (e.g., "Users are confused by my navigation")
3. **Click "JUMP TO SOLUTION!"** (or press Enter)
4. **Watch the magic happen** — the grid animates through a carousel of solutions
5. **Marvel at your newfound wisdom** — the game reveals the obviously correct fix

## The "Solutions"

The game randomly selects from an expertly curated list of tech industry's favorite quick-fixes:

- Add an AI agent
- Vibe code it
- Dark mode
- Put it in the backlog
- A/B test it
- Onboarding tour
- Build a dashboard
- Freemium model
- Notifications
- Social sharing
- Redesign the homepage
- Make it a setting

*Some solutions are weighted more heavily than others. "Add an AI agent" and "Vibe code it" are particularly aggressive.*

## Features

- **90s aesthetic** — complete with faux-3D buttons and grid textures that scream nostalgia
- **Weighted randomization** — because some terrible ideas are more popular than others
- **Keyboard support** — press Enter in the textarea to submit
- **Smooth animations** — animated cell highlighting and winner celebration
- **Mobile-friendly** — works great on desktop and tablet

## Technical Details

- **Pure vanilla HTML/CSS/JavaScript** — no frameworks, no dependencies
- **Web fonts** — Permanent Marker (for that handwritten feel) and Oswald (for bold tech vibes)
- **Responsive design** — scales from mobile to desktop
